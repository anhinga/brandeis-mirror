Because it is more and more difficult to run Java applets in a browser these days, the links in this page are not mirrored.

To follow them, switch to https://www.cs.brandeis.edu/~bukatin/neuro_simulations.html

WARNING: These experiments overrely on the Java promise that applets should not be able to harm the client machine. As a result, not enough attention is paid to the handling of the events related to closing of a window or a tab. Because Java actually breaks this promise, the client Web browser might crash when a tab or a window running one of these applets is being closed.
